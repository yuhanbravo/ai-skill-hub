## How to Read This Directory

- Treat `task_package` files as scoped task inputs only for the round that explicitly activates them.
- Treat `execution_report` and `review_report` files as historical evidence only; they do not authorize new work by themselves.
- Treat memo, summary, roadmap, and trigger-condition files as reference material only unless a current task package explicitly says otherwise.
- If this index conflicts with `AGENTS.md`, canonical skill guidance, or the active task package, follow those higher-priority sources.
